## Order Fulfilment Analysis

**Tools:** Python, Pandas, Matplotlib, Power BI  
**Dataset:** 1000 orders with shipping and profit data

### Key Findings
- Critical priority orders ship in ~1.7 days vs Low priority at ~6.4 days
- 20.9% of orders are delayed (>4 days)
- Delayed orders generate higher average profit (₹395 vs ₹260)
